# SideUI - Chrome Extension Sidebar

A lightweight Chrome extension that lets you save and manage your favorite URLs in a sidebar without storing any search history.

## What It Does

- 📝 **Save URLs** - Click a button to save any URL to your favorites
- 🔍 **Quick Search** - Search through your saved URLs instantly
- 📂 **Organize** - Keep all your favorite links in one easy-to-access sidebar
- 🔐 **Private** - No search history stored in Chrome, completely local
- ⌨️ **Keyboard Shortcut** - `Cmd+Shift+U` (Mac) or `Ctrl+Shift+U` (Windows/Linux)

## Features

- Sidebar interface for quick access
- Add URLs from popup or sidebar
- Open URLs in current tab or new tab
- Remove favorites anytime
- All data stored locally on your device
